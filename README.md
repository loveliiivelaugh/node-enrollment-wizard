# Starter Full-Stack Template

An opinionated, domain-agnostic full-stack template designed for rapid development with autonomous agents.

## Technologies

**Backend:**

- [Bun](https://bun.run/)
- [Hono](https://github.com/honojs/hono)

**Frontend:**

- [Typescript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/)
- [React](https://reactjs.org/)

## Quickstart

- Frontend: `cd frontend && pnpm install && pnpm dev`
- Backend: `cd backend && bun install && bun dev`

## Tests

- Frontend: `pnpm lint && pnpm test && pnpm build`
- Backend: `bun test`
